# My Books HUB — Design Plan

## Palette
- Oxblood primary: `#6B1D2A`
- Mustard secondary: `#C98A2B`
- Aged paper background: `#F7F0E3`
- Charcoal ink: `#241F1D`
- Saffron accent: `#D9712A`
- Warm ivory card: `#FFFDF8`

## Typography
- Headings: `Cormorant Garamond`, serif
- Body: `Inter`, sans-serif
- Utility / price labels: `IBM Plex Mono`, monospace

## Layout Signature
A library index-card look for each book entry: a card with a slightly raised paper feeling, a narrow label strip, and restrained hover lift.

## ASCII Wireframe
```
[Header: brand + quick links + CTA]
[Hero: story headline + trust badges + call-to-action]
[Stats strip]
[Catalog with filter tabs + search]
[Service cards: Retail / Buy-back / Rental / Festive]
[Rental calculator card]
[About founder + store story]
[Book Wiki search section]
[Contact form + contact links]
[Footer]
```

## Self-Critique
This direction avoids the usual generic bookstore template by leaning into warm paper, maroon tones, and a neighborhood-shop story rather than a startup-style SaaS look. The layout keeps the mobile-first structure while using a more tactile, familiar visual language.
